This page is for Operators & Conditional Statements in Java Script.

Comments in JS:


# Single line comment in JS:

WE can write a single line comment in Java Script by typing "//".

# Multiple line comment in JS:

We can write multiple line comment by typing "/*" at the starting of comment and "*/" at the end of comment.


Operators in JS:

Used to perform some operations in data. There are five types of operators in JS. (1) Arithmetic Operators, (2) Unary Operators, (3) Assignment Operators, (4) Comparison operators, (5) Logical Operators.

1. Arithmetic Operators:

  • +(plus), -(minus), *(multiplication), /(division)
  • Modulus: "%"
  • Exponentiation: "**"

2. Unary Operators:

  • Increment: "++". Example: a++
  • Decrement: "--". Example: a--

3. Assignment Operators:

  • "=", "+=", "-=", "*=", "%=", "**="

4. Comparison Operators:

  • Equal to: "=="
  • Equal to & type(strict): "==="
  • Not equal to: "!="
  • Not equal to & type(strict): "!=="
  • >(greater than), >=(greater thn and equal to), <(lower than), <==(lower than and equal to)

5. Logical Operators:

  • Logical AND "&&": If all conditions are true than it it will returns true otherwise false.
  • Logical OR "||": If only one condition comes true than it will returns true. If any condition doesn't come true than it will returns false.
  • Logical NOT "!": It always does the opposite. If any condition is true than it turns that into false and any condition is false than it turns that into true.


Conditional Statement:

There are three types of conditional statements. (1) if statement, (2) if-else statement, (3) else-if statement.
  • if statement: It can be written again and again after one if statement instead of else statement.
  • if-else statement: After if statement else statement can be written. Without if statement else can't be written and it can't be repeated again in one chain.
  • else-if statement: If there is lots of condition than we use else-if statement. First we write if than (else if) and in the end we write else statement. In this case we can completely skip the else statement in the end, the (else if) statement will do the else's work.
